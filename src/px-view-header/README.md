The `px.ViewHeader` component provides the user an overview of the content they are viewing and interacting with.

## Usage

```react
  <px.ViewHeader
    subtitle='Section Name'
    title='View Title'/>
```

## Usage

```react
const { view-header } = px;
<view-header>
  This is the children
</view-header>
```
